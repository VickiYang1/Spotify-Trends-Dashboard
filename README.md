# Spotify Trends Dashboard (PowerBI, Python)


## Overview
This project is an **interactive spotify analytics dashboard** built in **PowerBI**, using data retrieved from Spotify's developer web API.  
The dashboard provides insights into different artists from around the world, detailing information about their popularity ranking, genres, listeners, etc, enabling exploration of trends in the top 100 songs globally.

---

## Contents
- **Dataset:** `spotify_artists.csv`, `spotify_playlist_tracks.csv`
- **PowerBI Dashboard:** Interactive visualization with filters and calculated fields  
- **Mockups/Designs:** Reference layouts created before building in Canva 

---

## Features
- **Top Tracks Analysis:** Visualizes the Global Top 100 songs with metrics like popularity, genre, release date, and duration.  
- **Artist Breakdown:** Displays artist details including total songs in the playlist, popularity trends, follower counts, and genre distribution.  
- **Playlist Insights:** Compares track characteristics such as popularity ranges, explicit vs. non-explicit content, and overall playlist patterns.  
- **Trend Highlights:** Identifies rising artists, trending tracks, and shifts in listener engagement over time.  
- **Interactive Views:** Includes filters, drill-downs, and dynamic calculations for deeper, self-service exploration of Spotify data.  


---

## How to Use
1. Download the datasets `spotify_artists.csv` and `spotify_playlist_tracks.csv`
2. Open PowerBI.
3. Connect PowerBI to the dataset.
4. Import `spotify dashboard.pbix` into PowerBI
5. Explore the dashboard using interactive filters, tooltips, and data highlights.
